### What I learned

This was my very first Frontend Mentor challenge, and I learned a lot about the fundamentals:

- **Flexbox centering**: I used `display: flex`, `justify-content: center`, and `align-items: center` on the body to perfectly center the card both horizontally and vertically. Understanding `height: 100vh` was key to making it work.
- **The box model**: I discovered how `padding`, `margin`, and `border-radius` affect spacing and the overall look. Using `overflow: hidden` on the card helped contain the image’s sharp corners.
- **Responsive thinking**: I used `max-width` and relative units (`%`) so the card resizes gracefully on small screens. I also added `box-sizing: border-box` to handle padding correctly.
- **Styling text**: I applied Google Fonts, adjusted font weights, and used `text-align: center` to match the design.
- **Footer integration**: I experimented with placing the footer inside the card, which required modifying HTML structure and adjusting margins to keep everything balanced.
